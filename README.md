# NodeJS-ECommerce-API
Secure E-Commerce backend developed using Express.js and MongoDB Atlas. Includes user authentication, JWT-based authorization, encrypted passwords, product management, and scalable REST API architecture.
# E-Commerce Store Backend

A backend E-Commerce application built with Node.js, Express.js, MongoDB Atlas, JWT, and Bcrypt. Features secure user authentication, password encryption, product management, and REST APIs for handling users and products.

## Technologies Used
- Node.js
- Express.js
- MongoDB Atlas
- Mongoose
- JWT Authentication
- Bcrypt

## Features
- User Registration
- User Login
- JWT Token Authentication
- Add Products
- View Products
- MongoDB Database Integration
